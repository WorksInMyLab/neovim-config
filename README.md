# My Neovim Config
This repository contains my first Neovim config. I'm following along ThePrimeagen's Neovim tutorial [0 to LSP: Neovim RC From Scratch](https://www.youtube.com/watch?v=w7i4amO_zaE). 

## Why?
- I like vim
- VS Code is slow
- for fun and profit
